# مجلد tools/

ضع هنا ملف `smali.jar` من الإصدار الأخير.

## خطوات التحميل

1. افتح الرابط:  
   https://github.com/JesusFreke/smali/releases/latest

2. نزّل الملف المسمى **`smali-x.x.x.jar`**

3. أعد تسميته إلى **`smali.jar`**

4. ضعه في هذا المجلد `tools/smali.jar`

## تحقق سريع

```bash
java -jar tools/smali.jar --version
```

يجب أن تظهر رسالة مثل: `smali 2.5.2`

## ملاحظة

- يتطلب **Java 8** أو أحدث مثبتاً على السيرفر
- تحقق: `java -version`
